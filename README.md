# Y86-64

A reimplementation of the assembler `yas` and the simulator `yis` for Y86-64,
an educational instruction set architecture specified in
[CS:APP](http://csapp.cs.cmu.edu/).

## Usage

```
$ yas t.ys -o t.yo
$ yis t.yo
```
